从零复习Java之类和类之间的关系

发表于 2020-07-15

豆浆油条

欢迎关注豆浆的博客

类和类之间的关系

  • A is-a B 泛化（继承、实现）
  • A has-a B 包含（组合、聚合、关联）
  • A use-a B 依赖（依赖）

继承

  • 子类 父类
  • 实现：子类 extends 父类
  • 子类可以调用父类中的属性和方法（public和protected）
  • 子类可以添加自己独有的属性方法
  • 可以在子类中重写父类的方法（override）
  • 每个类都有继承类，如果不写extends关键字，默认基础Object，如果写了extends则继承后面那个父类。Object类没有父类
  • 只能单继承 可以通过传递的方式实现多继承，a extends b c extends b

    注意点

  • 方法重载（overload）与方法重写不一样，方法重载是一个类中的一组方法
  • 若是父类方法是final，子类无法重写，父类方法Static，子类不存在。父类方法是abstract，子类必须重写。而重载没有这些要求
  • 重载和重写的方法的名字规则一样，都是必须一致，参数，重写子类与父类的一致，而重载必须不一致
  • 如果父类方法抛出运行时异常，子类可以不予理会。如果父类抛出编译时异常，子类抛出异常的个数少于等于父类，子类抛出异常类型小于等于父类
  • 权限修饰符子类可以大于等于父类
  • 返回值类型子类可以小于等于父类
